The Standard Library

The standard library is a collection of modules written in C and built into the Pyro binary.

All standard library modules are submodules of the root module $std. This root module is imported automatically so you can access its submodules without using an explicit import statement, e.g. 

echo $std::math::pi;

Alternatively, you can explicitly import the submodule you want to use, e.g. 

import $std::math;

echo math::pi;
