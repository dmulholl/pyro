Syntax Highlighting

Vim

Pyro's git repository contains a syntax-highlighting file for Vim, etc/pyro.vim.

To enable syntax-highlighting in Vim, simply copy this file to your ~/.vim/syntax directory, e.g. 

cp ./etc/pyro.vim ~/.vim/syntax/

Visual Studio Code

You can find a syntax-highlighting extension for Pyro in VS Code's Extension Marketplace.

Pygments

You can find a Pygments lexer for Pyro here.

You can install this lexer directly from the Python package index by running: 

pip install pyro-lexer
